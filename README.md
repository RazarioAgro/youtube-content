
# YouTube — вёрстка интерфейса

Практический проект по вёрстке пользовательского интерфейса в стиле YouTube: 
главная страница, шапка с навигацией, сетка видео-карточек, адаптивная вёрстка.

## Технологии
- HTML5, CSS3
- Normalize.css
- SVG-иконки
- Vanilla JavaScript

## Что реализовано
- Адаптивная сетка карточек видео (grid/flexbox)
- Интерактивная навигация и шапка сайта
- Кроссбраузерная вёрстка, оптимизация под разные экраны
